Everything you need.

Powerful tools to manage your digital life securely.

Password Manager

Store and manage credentials securely with quick copy and autofill-ready structure. Everything stays on your device.

TOTP Generator (2FA)

Generate 6-digit codes for services like GitHub and Google with a built-in countdown timer. No separate app needed.

AES-GCM Encryption

All data is encrypted using the Web Crypto API (AES-GCM) with master password protection. Your data never leaves your device.

Manual Sync

Export your encrypted vault as a file and import it on another device anytime. You are in control of your backups.

Offline-First

Works completely offline. No accounts required, no risk of server breaches, and 100% privacy guaranteed.

Open Source

Transparent and auditable code. Trust in the code, not in a corporation. Join our community on GitHub.

Security by Design.

OPENLock is built on the principle that security tools should not require trust in a server — only trust in code.

  • AES-GCM 256-bit encryption for all sensitive data.
  • PBKDF2 key derivation with high iteration counts.
  • Zero-Knowledge architecture. We never see your data.
  • Local Storage only. No cloud syncing, no trackers.
